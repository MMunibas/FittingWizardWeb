Fitting Web 2.0.1: Help



Overview


The Fitting Wizard aims to provide a complete, automated workflow to obtain multipole moments and Lennard-Jones parameters that can subsequently be used in molecular dynamics simulations with the CHARMM software package. The process is divided into two independent stages, so that fitting of only multipole moments or only Lennard-Jones parameters is also possible. The two fitting processes are described in more detail in the corresponding subsections.

A copy of the fitting wizard software can be obtained from the GIT repository (https://github.com/MMunibas/FittingWizardWeb). You should download the latest branch and follow installation instructions on the group wiki: http://pc-studix0/wiki-meuwly/index.php/Group_Github#Web_Fitting_Environment. It runs as two separate components, ideally running on separate servers. The web UI contains the interface that is used to upload files, enter reference data and simulation parameters, store final files and display results. The API is designed to run on a server or cluster and handles job management, accepting files and new sessions from the UI and using python scripts supplied by developers in the Meuwly group to run the corresponding calculations, then return the results to the UI.

For more details on how you can set up a multipolar simulation in CHARMM once you have obtained your parameters, please see the CHARMM MTPL documentation. For scientific details on the fitting method or to cite this tool, please refer to the original publication (DOI: 10.1021/acs.jcim.6b00280).

Prerequisite experience: Note that the Fitting Wizard is designed to manage the many requisite Gaussian, CHARMM, GDMA and other calculations for you to greatly simplify the workflow, but you will still need a good working knowledge of these programs to obtain reliable results. Responsibility to interpret results correctly and check that generated input and output files are meaningful still lies with the user.
